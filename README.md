# Historia-Interactiva---Bertha-Hern-ndez-de-Ospina
Este proyecto es una historia interactiva digital sobre Bertha Hernández de Ospina, donde el usuario toma decisiones y observa sus consecuencias. El objetivo es enseñar la importancia de la educación y el impacto de esta líder en Colombia. Incluye una presentación final integrada para reforzar el aprendizaje.
